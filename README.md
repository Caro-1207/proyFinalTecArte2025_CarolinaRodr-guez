#Proyecto Final Arte

Para usar este codigo, es necesario instalar la libreria de python 'pandas' usando el siguiente comando: 

  sudo apt install python3-pandas

Después de haberla instalado, o en caso de ya haber tenido instalada dicha librería el código se puede ejecutar escribiendo:

  python3 figurasPrincipal.py

o en caso de tener una maquina virtual instalada y activada:

  python3 -m venv /home/user(en este caso 'happy')/proyectoFinalArte
